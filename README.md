# Yandex Wordstat

> Анализ поисковых запросов через Яндекс.Wordstat

## Зачем это нужно

Этот skill извлекает данные о поисковых запросах из Яндекс.Wordstat. Данные помогают:

- **Подбирать ключевые слова** для SEO и контекстной рекламы
- **Оценивать спрос** на товары и услуги
- **Находить похожие запросы** для расширения семантического ядра
- **Анализировать сезонность** запросов

## Что вы получите

Отчёты в форматах CSV и Markdown:

| Тип | Содержание |
|-----|------------|
| `freq` | Частотность запрошенной фразы |
| `similar` | Похожие поисковые запросы |
| `history` | Динамика показов (день/неделя/месяц) |

## Настройка

### 1. Создание OAuth-приложения

1. Перейдите на https://oauth.yandex.ru/client/new
2. Платформа: Веб-сервисы
3. Redirect URI: https://oauth.yandex.ru/verification_code
4. Доступы: Использование API Вордстата

### 2. Получение доступа к API

API Вордстата требует одобрения заявки. Подайте заявку по ссылке: https://yandex.ru/support2/wordstat/ru/content/api-wordstat (форма внизу страницы)

Понадобится:
- Ваш логин на Яндексе
- `ClientId` созданного приложения (найдёте на странице приложения)

Обычно одобрение приходит в течение нескольких часов.

### 3. Конфигурация

При первом запуске в текущей директории будет создан файл `wordstat_config.json`:

```json
{
  "client_id": "ВАШ_CLIENT_ID",
  "client_secret": "ВАШ_CLIENT_SECRET"
}
```

Заполните данные приложения.

### 4. Авторизация

При первом запуске скрипт запросит код авторизации.

## Использование

### Напрямую через PHP

```bash
# Базовые запросы
php .opencode/skills/yandex-wordstat/wordstat.php "купить ноутбук"
php .opencode/skills/yandex-wordstat/wordstat.php -t similar "seo"
php .opencode/skills/yandex-wordstat/wordstat.php -t history "маркетинг"

# С детализацией по времени
php .opencode/skills/yandex-wordstat/wordstat.php -t history -p daily -c 30 "opencode"
php .opencode/skills/yandex-wordstat/wordstat.php -t history -p weekly -c 8 "ai"
```

### Параметры

| Параметр | Сокращение | Описание | Пример |
|----------|------------|----------|--------|
| `--geo` | `-g` | ID региона | `-g 1` (Москва) |
| `--type` | `-t` | Тип отчёта | `-t similar` |
| `--limit` | `-l` | Лимит записей | `-l 20` |
| `--period` | `-p` | Период для history | `-p daily` |
| `--count` | `-c` | Количество периодов | `-c 30` |

### Типы периодов (для history)

| Значение | Описание | Ограничение |
|----------|----------|-------------|
| `daily` | По дням | Последние 60 дней |
| `weekly` | По неделям | С 2018 года |
| `monthly` | По месяцам (по умолчанию) | С 2018 года |
